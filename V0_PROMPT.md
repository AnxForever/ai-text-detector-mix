# V0提示词 - 中文AI文本检测演示前端

请帮我创建一个中文AI文本检测系统的演示前端页面。

## 功能需求

### 1. 文本输入区
- 大文本框，支持输入或粘贴中文文本
- 占位符提示："请输入或粘贴需要检测的中文文本..."
- 字数统计显示

### 2. 检测按钮
- 醒目的"开始检测"按钮
- 点击后显示加载动画

### 3. 结果展示区（分三部分）

#### 3.1 分类结果卡片
显示：
- 检测结果：Human / AI（大字体，带颜色）
- 置信度：百分比进度条
  - Human概率：绿色进度条
  - AI概率：红色进度条

#### 3.2 边界检测结果（如果是混合文本）
显示：
- 边界位置：第X字符
- 文本分段展示：
  - 人类部分：绿色背景高亮
  - AI部分：红色背景高亮

#### 3.3 详细信息
显示：
- 文本长度
- 检测类别（C2/C3/C4/Pure Human/Pure AI）
- 检测时间

### 4. 示例文本按钮
提供3个预设示例：
- 示例1：C2混合文本（人类开头+AI续写）
- 示例2：纯人类文本
- 示例3：纯AI文本

点击后自动填充到输入框

## 设计要求

### 风格
- 现代简洁的学术风格
- 使用渐变色和阴影
- 响应式设计（支持手机和电脑）

### 颜色方案
- 主色：蓝色系（#3B82F6）
- Human：绿色（#10B981）
- AI：红色/橙色（#EF4444）
- 背景：浅灰色（#F9FAFB）

### 布局
- 顶部：标题和说明
- 中间：输入区和检测按钮
- 底部：结果展示区（初始隐藏）

## 技术栈
- React + TypeScript
- Tailwind CSS
- Shadcn/ui组件
