## 数据集特点

✅ **高质量** - 平均质量分 0.998（接近完美）
✅ **长文本** - 平均长度 1,777 字符
✅ **多样化** - 六维组合生成（属性×话题×文体×角色×风格×约束）
✅ **已清洗** - 移除AI模板词汇，筛除低质量文本
✅ **均衡分布** - 五大属性均衡覆盖

## 数据字段说明

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `text_id` | String | 文本唯一标识符 |
| `text_content` | String | 文本内容（已清洗） |
| `source_api` | String | 生成来源API |
| `source_model` | String | 使用的模型名称 |
| `attribute` | String | 文本属性（议论/说明/描写/记叙/抒情） |
| `topic` | String | 主题内容 |
| `genre` | String | 文体类型 |
| `role` | String | 角色设定 |
| `style` | String | 写作风格 |
| `constraint` | String | 创作约束 |
| `prompt` | String | 生成提示词 |
| `combination_quality` | Float | 组合质量分 |
| `generation_quality` | Float | 生成质量分（0-1） |
| `length` | Integer | 文本长度（字符数） |
| `timestamp` | String | 生成时间戳 |

## 使用方法

### Python 加载

```python
import pandas as pd

# 加载数据集
df = pd.read_csv('parallel_dataset_cleaned.csv', encoding='utf-8-sig')

# 查看基本信息
print(f"总记录数: {len(df)}")
print(f"平均质量: {df['generation_quality'].mean():.3f}")
print(f"平均长度: {df['length'].mean():.0f} 字符")

# 按属性筛选
df_argumentation = df[df['attribute'] == '议论']
print(f"议论类文本: {len(df_argumentation)} 条")

# 按质量筛选
df_high_quality = df[df['generation_quality'] >= 0.9]
print(f"高质量文本: {len(df_high_quality)} 条")

# 查看样本
print(df.head())
```

## 数据质量保证

### 清洗步骤
1. ✅ 删除AI模板词汇（首先、其次、最后、综上所述等）
2. ✅ 删除AI特征短语（作为AI、我是一个、很抱歉等）
3. ✅ 筛除长度 <300字 的文本
4. ✅ 筛除质量 <0.7 的文本
5. ✅ 清理多余空格和空行

- **保留率:** 92.1%（9,170/9,956）
- **平均质量提升:** 0.973 → 0.998
- **平均长度提升:** 1,647 → 1,777 字符
