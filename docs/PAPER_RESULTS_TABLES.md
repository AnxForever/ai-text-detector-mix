## 表2：格式偏差统计（去偏前）

| 数据集 | AI Markdown率 | 人类 Markdown率 | 格式偏差 | 简单规则准确率 |
|--------|--------------|----------------|----------|--------------|
| 训练集 | 63.81% | 0.04% | **63.77%** | 81.00% |
| 验证集 | 63.70% | 0.10% | **63.60%** | 80.91% |
| 测试集 | 65.00% | 0.10% | **64.90%** | **81.61%** |
| 平均 | 64.17% | 0.08% | **64.09%** | 81.17% |

**关键发现**：
- ⚠️ 严重格式偏差（>60%）
- ⚠️ 简单规则（仅判断markdown）准确率>80%
- ⚠️ 模型可能学习格式而非语义

## 表3：格式去偏效果对比

| 指标 | 训练集 | 验证集 | 测试集 |
|------|--------|--------|--------|
| **去偏前** | | | |
| AI Markdown率 | 63.81% | 63.70% | 65.00% |
| 人类 Markdown率 | 0.04% | 0.10% | 0.10% |
| 格式偏差 | 63.77% | 63.60% | 64.90% |
| 简单规则准确率 | 81.00% | 80.91% | 81.61% |
| **去偏后** | | | |
| AI Markdown率 | 2.37% | 1.73% | 2.51% |
| 人类 Markdown率 | 0.04% | 0.10% | 0.10% |
| 格式偏差 | **2.33%** | **1.63%** | **2.41%** |
| 简单规则准确率 | **48.76%** | **48.39%** | **48.87%** |
| **改善幅度** | | | |
| 格式偏差降低 | ↓ 61.44% | ↓ 61.97% | ↓ 62.49% |
| 简单规则准确率降低 | ↓ 32.23% | ↓ 32.52% | ↓ 32.74% |

**结论**：
- ✅ 格式偏差从64%降至<3%（减少97%）
- ✅ 简单规则准确率降至<50%（接近随机猜测）
- ✅ 成功消除格式作为有效特征
